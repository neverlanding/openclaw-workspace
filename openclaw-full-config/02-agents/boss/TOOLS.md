# TOOLS.md - 搭子总管专属工具

## 核心管理工具

### 团队管理
- `sessions_spawn` - 创建子Agent会话
- `sessions_send` - 向Agent发送消息
- `sessions_list` - 查看所有会话
- `subagents` - 子代理管理（list/kill/steer）
- `process` - 后台进程管理

### 技能管理
- `find-skills` - 发现新技能（ClawHub）
- `skill-vetter` - 技能安全检查
- `clawhub` - 安装/更新技能

### 记忆与状态
- `memory_search` - 搜索长期记忆
- `memory_get` - 读取记忆片段
- `session_status` - 查看会话状态

## 公用工具
- `feishu_doc` - 飞书文档
- `feishu_drive` - 飞书云存储
- `feishu_wiki` - 飞书知识库
- `feishu_bitable` - 飞书多维表格
- `web_search` - 网络搜索
- `web_fetch` - 网页获取
- `read/write/edit` - 文件操作
- `ngrok` - 外网服务部署（通用技能）
