# 失败教训速查

*真实错误记录 - 防止重复犯错*

*版本: v1.0 | 更新: 2026-03-19*

## 🚨 已验证错误（5个）

### 1. 子Agent超时
**症状**: 任务运行超1分钟后失败，无结果  
**预防**: 复杂任务拆分；超时=预估×1.5；超时后提供部分结果

### 2. 网络代理问题
**症状**: 多个API调用失败，错误信息不明确  
**预防**: 任务前检查网络连通性；建立网络诊断流程

### 3. 文件路径混乱
**症状**: /tmp文件未迁移到工作区，导致丢失  
**预防**: 项目文件统一放工作区；创建后检查清单

### 4. 任务依赖冲突
**症状**: 并行任务修改同一文件，产生冲突  
**预防**: 分析依赖关系；相关任务串行；独立才并行

### 5. 缺少用户确认
**症状**: 自动执行重要操作，导致意外结果  
**预防**: 定义重要操作清单；执行前确认；提供预览

### 6. 飞书文档内容为空
**症状**: 发送的文档链接打开后只有标题，无正文内容  
**原因**: 创建文档时只写了标题，未执行追加内容操作  
**解决**: 立即追加完整内容，重新发送  
**预防**: 
- 创建文档后立即执行 `feishu_doc read` 验证内容
- 发送前再次确认文档内容完整
- 重要文档本地备份+飞书文档双重保存
