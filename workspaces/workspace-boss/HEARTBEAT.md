# HEARTBEAT.md - 搭子总管心跳任务清单

## 触发方式
由外部 Cron 每30分钟触发一次，本文件定义智能检查任务。

## 定时任务（由 Cron 每30分钟触发）

### 智能检查（选择性执行，每次选2-3项）

#### 1. 邮件检查
- [ ] 检查未读邮件
- [ ] 标记重要邮件
- [ ] 发现紧急邮件时立即提醒组长

#### 2. 日历检查
- [ ] 检查24小时内即将到来的事件
- [ ] 提醒2小时内的事件

#### 3. 提醒任务检查
- [ ] 查看即将执行的提醒
- [ ] 检查是否有需要通知组长的即将到来的提醒

#### 4. 助手状态检查
- [ ] 检查各助手是否有待汇报事项
- [ ] 汇总需要组长关注的信息

## 记忆维护任务（深夜执行，23:00后）

### 每日
- [ ] 确保当日日志文件存在
- [ ] 检查 memory/appointments.md 中的待办事项
- [ ] 更新 MEMORY.md（如有重要事项）

### 每周（周一）
- [ ] 回顾上周日志，提取关键信息
- [ ] 更新 lessons/ 目录
- [ ] 清理过期临时文件

## 执行原则
1. **选择性执行**: 每次心跳选择2-3个相关任务
2. **时间感知**: 23:00-08:00 除非紧急，保持安静
3. **记录结果**: 在 memory/YYYY-MM-DD.md 中记录检查发现的行动项
