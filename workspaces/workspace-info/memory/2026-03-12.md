  --name "news-daily-brief-v3" \
  --description "每日早间科技简报" \

  --tz "Asia/Shanghai" \

  --session isolated \
  --no-deliver \
  --timeout-seconds 300

## ✅ 正确的定时任务配置（已验证）

### 飞书消息推送正确格式
```bash
openclaw cron add \
  --name "news-daily-feishu" \
  --description "每日早间简报飞书推送" \
  --cron "0 9 * * *" \
  --tz "Asia/Shanghai" \
  --agent news \
  --message "生成早间科技简报并推送给组长" \
  --session isolated \
  --announce \
  --channel feishu \
  --to "user:ou_dc6495121ba8ee1ecbd6f9a2c123eb0c" \
  --timeout-seconds 300
```

### 关键参数说明
| 参数 | 作用 |
|:---|:---|
| `--session isolated` | 在隔离session中执行 |
| `--announce` | 发送执行结果给用户 |
| `--channel feishu` | 飞书渠道 |
| `--to "user:xxx"` | 目标用户ID |
| `--timeout-seconds 300` | 5分钟超时 |

### 理财搭子配置参考
- 同样的 `--announce --channel feishu --to` 格式
- `--session isolated` 隔离执行
- 消息内容在 `--message` 中指定
