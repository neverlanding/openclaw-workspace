## 核心规则（必须遵守）

### 1. 超时设置公式
```
超时时间 = 视频时长(分钟) × 2
```
**示例**: 162分钟视频 → 324分钟 = 5.4小时 → **设置5.5小时(19800秒)**

### 2. 时长验证要求
必须验证三者一致性：
- [ ] 视频时长（B站页面）
- [ ] 音频时长（ffprobe检查）
- [ ] 转录文本覆盖率（字符数估算）

**覆盖率标准**: ≥95%

### 3. 进度汇报频率
- **每30分钟**主动汇报一次
- 关键节点立即汇报（下载完成、转录完成等）
