## Key Learnings

### Memory System
- `~/.openclaw/memory/*.sqlite` files = Vector embedding indexes for semantic search
- One SQLite per Agent (boss.sqlite, coder.sqlite, etc.)
- Stores text chunks + vector embeddings from MEMORY.md and memory/*.md
- Enables `memory_search` tool to find relevant context

### Browser Data Safety
- **Safe to delete**: Cache/, Code Cache/, optimization_guide_model_store/

- Browser cache cleanup does NOT affect running web services (Star-Office-UI, Clawalytics)
