# 公共知识库使用指南

## 📚 什么是公共知识库

**位置**: `~/.openclaw/shared-knowledge/`
**维护者**: 搭子总管
**用途**: 8个助手共享的通用信息

## 📂 包含内容

| 文件 | 内容 | 何时查阅 |
|------|------|----------|
| `SKILL_SOURCES.md` | 技能查找渠道 | 用户要找技能时 |
| `USER_MAPPING.md` | 组长ID映射表 | 给组长发消息前核对 |
| `COMMON_RULES.md` | 通用规则 | 不确定规范时 |

## 🔍 何时主动读取

**必须查阅公共知识库的场景：**

1. **查找技能时** → 先看 `SKILL_SOURCES.md`
2. **给组长发消息前** → 确认流程（`COMMON_RULES.md`）
3. **需要修改Gateway配置/重启时** → 必须查 `COMMON_RULES.md` 的"Gateway重启安全机制"
4. **创建定时任务时** → 必须查 `COMMON_RULES.md` 的"定时任务规范"
5. **不确定某个规则时** → 查 `COMMON_RULES.md`

## ❓ 如何使用

```bash
# 方法1: 直接读取文件
read ~/.openclaw/shared-knowledge/SKILL_SOURCES.md

# 方法2: 向总管询问
"总管，帮我查一下公共知识库里的XXX"
```

## 📝 注意事项

- ✅ 公共知识由**总管统一维护**
- ❌ 助手**不要自行修改**公共知识文件
- ✅ 发现公共知识有误 → **通知总管更新**

---
**记住**: 公共知识库是大家的共享资源，善用它！
